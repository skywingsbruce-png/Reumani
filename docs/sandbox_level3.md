# 沙箱 Level 3（容器强隔离）规划

Level 2（现状，`ssc_sandbox.py`）= 静态扫描 + 环境剥密钥 + 目录限定 + 一任务一目录，**不是真隔离**，
**只适用于公开、非敏感科研数据**。患者数据/敏感数据必须等 Level 3。

## Level 3 硬要求
- **一任务一容器**（run 结束即销毁）。
- **非 root** 运行。
- **只读输入**卷；**独立输出**卷（不与其它任务共享）。
- **默认断网**；仅按**域名白名单**放行（如 ebi.ac.uk / ncbi.nlm.nih.gov）。
- **资源限额**：CPU、内存、磁盘、墙钟时间（cgroups / --memory / --cpus / ulimit）。
- **固定镜像 + 镜像 digest 锁定**（`image@sha256:...`），不可漂移。
- **完整日志**（stdout/stderr/退出码/资源用量/网络请求）留存审计。
- **运行结束销毁容器**（`--rm`）。

## Level 2 → Level 3 迁移接口
`safe_run_python(code, timeout)` 的签名保持不变；Level 3 实现替换其内部（改为在容器里跑），
上层（工具/A1）无需改动。切换由配置/环境变量决定，默认仍 Level 2（向后兼容）。
