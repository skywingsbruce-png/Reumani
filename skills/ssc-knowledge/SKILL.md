# SSc 专科知识底座

这是一份供 agent 随时查阅的 SSc（系统性硬化症）领域常识手册。用它来判断分型、解读抗体、
搭建研究背景，**而不是**每次都去现查。涉及真实患者的诊断/治疗决策，只做参考，由临床医生拍板。

## 1. 分类标准
- **ACR/EULAR 2013 分类标准**（评分制，总分≥9 分可分类为 SSc）：
  - 双手手指皮肤增厚延伸至掌指关节近端（充分条件，单独 9 分）
  - 手指皮肤增厚（肿胀指 2 / 硬指端 4）
  - 指端病变（指尖溃疡 2 / 指凹陷性瘢痕 3）
  - 毛细血管扩张（2）
  - 甲襞毛细血管异常（2）
  - 肺动脉高压和/或间质性肺病（2）
  - 雷诺现象（3）
  - SSc 相关自身抗体（抗着丝点 / 抗 Scl-70 / 抗 RNA 聚合酶 III，任一 3 分）
- 早期识别常用 **VEDOSS**（超早期 SSc）概念：雷诺 + 抗体 + 甲襞异常。

## 2. 亚型（决定预后与随访重点）
- **lcSSc（局限型）**：皮肤增厚限于肘/膝远端和面部；常抗着丝点阳性；肺动脉高压(PAH)风险高，进展慢。
- **dcSSc（弥漫型）**：皮肤增厚累及近端肢体/躯干；常抗 Scl-70 或抗 RNAP III 阳性；早期 ILD、肾危象风险高，进展快。
- 皮肤评分用 **mRSS（modified Rodnan Skin Score，0-51）**。

## 3. 自身抗体 → 临床表型关联（SSc 的"分子分型"精髓）
| 自身抗体 | 亚型倾向 | 高风险表型 |
|---|---|---|
| 抗着丝点 (ACA) | lcSSc | 肺动脉高压(PAH)、进展慢 |
| 抗 Scl-70 (抗拓扑异构酶I) | dcSSc | 间质性肺病(ILD) |
| 抗 RNA 聚合酶 III | dcSSc | 硬皮病肾危象(SRC)、快速皮肤进展、与肿瘤相关 |
| 抗 U3-RNP (fibrillarin) | dcSSc | PAH、心肌、肌炎 |
| 抗 Th/To | lcSSc | ILD、PAH |
| 抗 PM-Scl | 重叠综合征 | 肌炎重叠 |

## 4. 器官受累谱（随访要点）
- **肺**：ILD（最常见死因，HRCT + 肺功能 FVC/DLCO 监测）、PAH（右心导管确诊）。
- **皮肤**：mRSS 动态评估。
- **血管**：雷诺、指端溃疡、甲襞毛细血管镜（早/活动/晚期模式）。
- **肾**：硬皮病肾危象（恶性高血压 + 急性肾损伤，抗 RNAP III、大剂量激素为危险因素，ACEI 是关键治疗）。
- **消化道**：食管动力障碍、胃窦血管扩张(GAVE)。
- **心脏**：心肌纤维化、传导异常。

## 5. 治疗要点（EULAR 推荐方向，供研究背景用）
- ILD：吗替麦考酚酯、环磷酰胺、**尼达尼布**（抗纤维化）、托珠单抗（早期炎症型）。
- 皮肤/早期 dcSSc：MMF、甲氨蝶呤；重症可考虑**自体造血干细胞移植(HSCT)**。
- 血管：钙拮抗剂、PDE5 抑制剂、前列环素类、波生坦（指端溃疡/PAH）。
- 肾危象：ACEI（禁用大剂量激素诱发）。
- 前沿：**CD19-CAR T**、双特异性 T 细胞衔接器(BiTE)（研究阶段）。

## 6. 发病机制三大轴（研究切入点）
1. **免疫失调**：自身抗体、T/B 细胞、干扰素特征、巨噬细胞。
2. **血管病变**：内皮损伤、血管稀疏、缺氧。
3. **纤维化**：肌成纤维细胞持续活化，TGF-β/Wnt/YAP 通路，细胞外基质沉积。
- 单细胞/空间组学揭示皮肤、肺中致病性成纤维细胞亚群与免疫-基质互作是当红方向。

## 7. SSc 与染色体不稳定(CIN)/衰老（本课题组的差异化角度，见 [[ssc-cin]] 技能）
- 成纤维细胞衰老、DNA 损伤反应、复制应激可能参与持续纤维化。
- CIN（染色体不稳定）相关的基因组不稳定、微核、cGAS-STING 天然免疫激活，
  是连接"基因组不稳定 → 天然免疫 → 纤维化"的潜在机制，值得在 SSc 组学数据中检验。
- 分析层面见 `ssc-cin` 技能（CIN signature 打分、与临床/纤维化表型关联）。

## 边界
- 本手册是研究与写作的背景知识，**不构成对任何具体患者的诊疗建议**。
- 数据要以最新指南和原始文献为准；写论文时关键结论需回到原文献核对。
