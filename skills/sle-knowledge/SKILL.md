# SLE 专科知识底座

供 agent 查阅的 SLE 常识手册。涉及真实患者诊疗只作参考，由临床医生拍板。

## 分类标准
- **ACR/EULAR 2019**：入门条件 **ANA≥1:80**；随后按 7 个临床域 + 3 个免疫域加权评分，**总分≥10** 可分类。
- 强调：ANA 阴性基本可排除 SLE（高敏感性）。

## 关键自身抗体 → 临床关联
| 抗体 | 关联 |
|---|---|
| 抗 dsDNA | 疾病活动、**狼疮肾炎**；可随病情波动 |
| 抗 Sm | SLE 特异性高 |
| 抗磷脂抗体(aPL) | 血栓、流产、抗磷脂综合征 |
| 抗 Ro/SSA、La/SSB | 干燥重叠、新生儿狼疮/先天心脏传导阻滞 |
| 抗 U1-RNP | 混合结缔组织病 |
| 低补体(C3/C4↓) | 活动、狼疮肾炎 |

## 器官受累
- **肾（狼疮肾炎）**：最重要预后决定因素；ISN/RPS 病理 I–VI 型，III/IV 型需强化免疫抑制。
- 皮肤黏膜（颊部红斑、盘状、光敏、口腔溃疡）、关节、血液（三系减少）、浆膜炎、神经精神狼疮(NPSLE)。

## 治疗要点
- **羟氯喹**：几乎所有患者的基础用药（降复发、护器官）。
- 激素 + 免疫抑制（MMF、环磷酰胺）——狼疮肾炎诱导/维持。
- 生物制剂：**贝利尤单抗(belimumab, 抗 BAFF)**、**anifrolumab(抗 I 型干扰素受体)**；狼疮肾炎可加 **伏环孢素(voclosporin)**。
- 前沿：**CD19-CAR T**（重症难治）。

## 发病机制（与你 CIN/干扰素课题强相关）
- **I 型干扰素特征(IFN signature)** 是 SLE 核心——anifrolumab 直接靶向它。
- **核酸感应**：凋亡细胞清除障碍 → 胞外/胞质核酸 → **TLR7/9 + cGAS-STING** → I 型干扰素 → 自身免疫。
- **TREX1、DNASE1L3**（降解 DNA 的酶）功能缺失是单基因狼疮病因——直接把"DNA 处理异常→干扰素"钉死为机制。
> 这正是把你 SSc 的 cGAS-STING/干扰素角度横向扩到 SLE 的天然接口：SLE 里"核酸→cGAS-STING→IFN"证据比 SSc 强得多，是跨病共性机制的好切入点。
