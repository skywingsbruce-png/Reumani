# Know-how：组学常见混杂和统计规范

做 SSc 转录组/单细胞分析时必须守的统计规范，避免假阳性。

## bulk RNA-seq
- **批次效应**：不同测序批次/中心会造成假差异。设计时随机化，分析时校正（如把 batch 放进模型）。
- **归一化**：原始 count 用 DESeq2/edgeR；已归一化的 log 值可用 t/limma 类方法。别混用。
- **多重检验**：几万个基因要做 FDR 校正（BH），只看原始 p 会大量假阳性。
- **协变量**：年龄、性别、病程、用药可能混杂，纳入模型或分层。

## 单细胞 scRNA-seq
- **供体级重复最重要**：多个样本的细胞**不是独立样本**。比较组间差异要做 **pseudobulk**（把每个供体每种细胞聚合成一个样本再比），否则 n 被极度虚高、假阳性爆炸。
- **批次/整合**：多样本需整合（Harmony/scVI）去批次，否则聚类是按样本分而不是按细胞类型分。
- **双细胞(doublet)**：过滤，否则出现假的"中间态"细胞。
- **注释**：自动聚类只是起点，细胞类型要结合 marker + 领域知识人工确认。

## 通用
- **相关≠因果**。
- **效应量 + CI** 比单看 p 值更有信息。
- **可复现**：记录随机种子、软件版本、参数（见数据湖/可复现记录）。
- **过拟合**：小样本 + 高维特征，分类器要交叉验证 + 独立验证集。

## SSc 特有提醒
- 皮肤取材部位/病程要匹配。
- 原代成纤维细胞传代数要一致。
- 样本量普遍偏小，警惕"显著但不稳"。
