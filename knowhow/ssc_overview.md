# Know-how：SSc 研究领域概览

系统性硬化症（SSc）研究的核心逻辑与当前热点，供 agent 规划研究方向时参考。

## 三大发病轴（任何 SSc 研究都绕不开）
1. **免疫失调**：自身抗体、T/B 细胞、I 型干扰素特征、巨噬细胞极化。
2. **血管病变**：内皮损伤、毛细血管稀疏、缺氧、血管新生异常。
3. **纤维化**：肌成纤维细胞持续活化，TGF-β / Wnt / YAP-TAZ 通路，ECM 沉积。
> 三条轴互相耦合，别当成独立模块——免疫-血管-纤维化是级联。

## 亚型与预后（决定研究人群选择）
- **lcSSc**：抗着丝点多见，PAH 风险高，进展慢。
- **dcSSc**：抗 Scl-70 / 抗 RNAP III 多见，早期 ILD、肾危象风险高，进展快。
- 早期 dcSSc 是很多干预（含细胞治疗）研究的目标人群。

## 当前热点方向（近年高影响力集中处）
- **单细胞/空间组学**：皮肤、肺的致病性成纤维细胞亚群（SFRP2+/PRSS23+ 等）、免疫-基质互作。
- **内在基因表达亚型**：inflammatory / fibroproliferative / limited / normal-like（用于患者分层）。
- **抗纤维化**：尼达尼布及新靶点；成纤维细胞去分化/衰老。
- **细胞治疗**：CD19-CAR T、BiTE（研究阶段，注意安全性与人群选择）。
- **本课题组差异化角度**：染色体不稳定(CIN)/DNA 损伤/cGAS-STING → 纤维化（见 ssc-cin 技能）。
