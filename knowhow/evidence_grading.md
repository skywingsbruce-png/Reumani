# Know-how：生物医学证据分级

judge / verify_claim 判断证据强度时的统一标准。

## 研究设计层级（从强到弱，粗略）
1. 系统综述 / Meta 分析（高质量 RCT 汇总）
2. 随机对照试验（RCT）
3. 前瞻性队列
4. 回顾性队列 / 病例对照
5. 横断面研究
6. 病例系列 / 个案报道
7. 基础实验（细胞/动物）—— 机制强，但**人体因果证据弱**
8. 专家意见 / 综述观点

> 在 SSc：很多"突破性"结果是**单臂、小样本概念验证**或**动物/细胞实验**，机制价值高但因果证据等级不高。

## 打分要看的维度
- **样本量**：N 多大？分组后每组多少？小样本(<30/组)结论不稳。
- **对照**：有没有健康/疾病对照？单臂无对照要谨慎。
- **随访**：横断面不能推因果；纵向随访多长？
- **重复**：独立队列验证了吗？供体级重复（单细胞）？
- **效应量 + 置信区间**：不只看 p，看效应大小和 CI 宽窄。

## 过度解读的典型信号（verify 时重点抓）
- 把**相关**说成**因果**。
- 把**小样本/单臂**说成"证实"。
- 把**动物/细胞**结论直接外推到**病人**。
- 把**统计显著**等同于**临床意义**。
- 把**替代指标**（如 signature score）当成**硬终点**。

## 输出建议
给每条结论标：证据等级(high/moderate/low) + 一句理由 + 支持它的具体文献。
